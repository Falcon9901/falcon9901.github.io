How to Deploy Django Applications on Heroku using Python Decouple

Shaaah.in
2018-01-19
Django, Heroku, Python Decouple

Prerequisites

You should have a local Django app setup using Postgres setup neatly in a virtualenv

  • Python development environment
  • local Django application using PostgreSQL
  • Heroku Account
  • the Heroku CLI installed locally

Install dj-database-url

Install psycopg2

Install Python Decouple

for static file serving its highly recomended using django Storage package

Preparing the Application

  • Add a Procfile in the project root;
  • update requirements.txt file with all the requirements in the project root;
  • Add Gunicorn to requirements.txt;
  • A runtime.txt to specify the correct Python version in the project root;
  • A .env file to store database credential in the config directory;
