Django

Prerequisites

  • Python development environment

for Database its highly recomended using PostgreSQL

Most problems usually can’t be discovered until the project is run on a strongly typed database (e.g. PostgreSQL or MySQL).
When these types of bugs hit, you end up kicking yourself and
scrambling to setup your local development machine with the right database.

Two Scoops of Django

Installing Django

install Django with pip3

make a directory for your project and navigate to the directory

optional : you can make a notes.txt

craete requirements.txt

create a project

Creating the sample application

make a directory for your apps and navigate to the directory

add your app to INSTALLED_APPS on config/settings.py

Running database migrations

run the development web server from Project folder

note : CTRL+C to stop development server
view the site by navigating to http://127.0.0.1:8000/ in your local web browser.
